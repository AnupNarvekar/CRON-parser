This CLI application parses a standard cron string (minute, hour, day of month, month, and day of week + command) and expands each field to show the times at which it will run.

This repo was created as an assignment for HOI

# Installation & setup
## Requirements:
- Node.js

## Usage:
1. Clone/download this repo
2. Open a terminal in this repo
3. Run: node cron-parser.js <CRON STRING>

Note: Replace <CRON STRING> with appropriate value for your choice.
